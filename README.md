# parser-toolbox

## Install
**Add to composer.json**
```
    "repositories": [
        {
            "type": "vcs",
            "url":  "https://github.com/bezdelnique/parser-toolbox"
        }
    ],
```

**Run**
```
composer require bezdelnique/parser-toolbox
```


## Force project update
```
rm -Rf vendor/bezdelnique/parser-toolbox
composer install
```
